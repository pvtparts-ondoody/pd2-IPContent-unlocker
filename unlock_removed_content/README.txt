Unlocks as much removed content as I could find in a few hours of looking; pretty sure it has the most important ones.


Technical trivia:
Everything referred to as a bundle or pack should also include all related weapons, melees, perk decks, masks, etc... if you find otherwise, let me know.
A correlation of which DLC listings in the .lua file unlock which things:

"berry" = Point Break heists DLC bundle.
"chico" = Scarface character DLC pack.
"mad" = Hardcore Henry heists bundle.
"coco" = Jimmy character pack.
"pim" = John Wick Weapon Pack DLC.
"spa" = John Wick heists DLC bundle.
"rvd" = Presumably originally intended for the Reservoir Dogs heist bundle & associated items, which it provides. Interestingly also provides Bodhi, and his related sniper rifle (Platypus 70). I can only assume this is because when they created the "PAYDAY 2: IP Content" steam DLC for pre-2024 players, Bodhi (who otherwise has no character pack DLC listing of his own) had to be removed too, and so was merged into this listing. This also leads me to believe that the previously mentioned "PAYDAY 2: IP Content" DLC specifically enables the rvd, john_wick_character, coco, and mad DLC listings: they're all FREE IP additions, as opposed to the paid Point Break DLC bundle or Scarface.
"bobblehead" = Rewards the 'Senpai-Dozer" mask that was originally obtainable by buying a physical dozer bobblehead from OVERKILL. As STARBREEZE no longer sells these, it's now unobtainable normally.
"john_wick_character" = John Wick character pack.
"sparkle" = Provides the Hoxton's Housewarming Party Hat mask. As of writing, still technically legitimately obtainable in some regions via this link:
	https://www.paydaythegame.com/static/payday2/social-safe/housewarming/
